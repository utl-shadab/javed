# TILA - Frontend Template

This is a frontend-only Next.js + TypeScript starter built for *The Indian Legal Associates*.
It includes TailwindCSS, GSAP, Framer Motion, and Lenis for smooth scrolling, plus PWA config (next-pwa).

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build:
```bash
npm run build
npm run start
```

Note: This repo is frontend-only (per your request). No backend implementation included.
